Greenden_tailwind

A clean, responsive multipage plant-themed website built using HTML and Tailwind CSS.
It includes a Home page, Products page, and Contact page—each styled with Tailwind utility classes and designed for a smooth user experience.

Project Overview :

Greenden_tailwind is a front-end practice project focusing on layout design, component styling, and responsive UI using Tailwind CSS.
It includes:

✔ Home Page (index.html) :

Hero section with heading, buttons, and plant image

“About Us” highlights

Best-seller grid with hover animations

Customer reviews section

Newsletter subscription input

Footer section

✔ Products Page (products.html) :

Search bar for product lookup UI

Grid-based display of plant items

Hover animations on product images

Product name + price display

Shared responsive navbar + footer

✔ Contact Page (contact.html) :

Clean contact form (email, subject, message)

Tailwind forms with responsive design

Shared navbar and footer

Tech Stack :

HTML5

Tailwind CSS (CDN version 2.2.19)

Responsive design powered by utility classes

No JavaScript required
